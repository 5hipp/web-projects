HTML Validation

HTML Validation is mainly about meticulously following syntactical rules.

  • The first validation can be by a programmers editors such as Bracket. Note, the spelling mistake, navs id highlighted in red (it should be nav)
  • Leave the mis-spelt navs and save the file. Using the operating system file browser find html5-validation.html file and drag it into Firefox browser (or in Firefox, use File > Open). Even with the error, the content is displayed; browsers are very forgiving and correct the code as far as possible. Right-mouse click and select view page source and note the errors in red.
  • The third method is to use the W3C Validation service. Select Validate by File Upload or cut& the HTML source code into Validate by Direct Input and Check for an error report.
